# Тест Cleverence C#

## Задание
- Ссылка на задание [link](Cleverence/task/)
- Разрабатывалось на Ubuntu, VSCode, dotnet10.

## Реализация

- Проект в папке Cleverence. Файлы для каждого номера задания в папках task1, task2, task3.
- Тесты к заданиям в папке Test тестового проекта xUnit.

## Запуск

- task1 смотрим в папку ```Cleverence/task1``` и запускаем тесты ```Test/StringCompressorTests.cs```
- task2 смотрим в папку ```Cleverence/task2``` и запускаем тесты ```Test/ServerTests.cs```
- task3 смотрим в папку ```Cleverence/task3``` и запускаем проект Cleverence. В результате заполнятся файлы в папке test3. И запускаем тесты ```Test/LogTests.cs```
- Запустив проект Test как тесты, все тесты для всех заданий сразу отработают.
