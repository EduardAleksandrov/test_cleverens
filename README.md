# Тест Cleverens C#

## Задание
- Ссылка на задание [link](Cleverens/task/)
- Разрабатывалось на Ubuntu и VSCode.

## Реализация

- Проект в папке Cleverens. Файлы для каждого номера задания в папках task1, task2, task3.
- Тесты к заданиям в папке Test тестового проекта xUnit.

## Запуск

- task1 смотрим в папку Cleverens/task1 и запускаем тесты Test/StringCompressorTests.cs
- task2 смотрим в папку Cleverens/task2 и запускаем тесты Test/ServerTests.cs
- task3 смотрим в папку Cleverens/task3 и запускаем проект Cleverens. В результате заполнятся файлы в папке test3. И запускаем тесты Test/LogTests.cs
- Запустив проект Test как тесты, все тесты для всех заданий сразу отработают.
